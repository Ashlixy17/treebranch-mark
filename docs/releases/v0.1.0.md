# Treebranch Mark v0.1.0

`v0.1.0` 是 Treebranch Mark 从 Alpha 进入第一个可用版本的发布。相比 `v0.1.0-alpha`，本次重点放在可用性、稳定性和第一张真正可展示的图上。

## Highlights

- 新增 Source 层 Memory Snapshot Cache，减少重复 GitHub API 请求。
- SVG Commit 节点支持显示 GitHub 贡献者头像。
- 头像不可用时自动回退为圆形 Commit 节点。
- 增强头像 URL 处理，避免空头像地址和特殊字符破坏 SVG。
- Browser Viewer 可以直接展示完整 Pipeline 生成的 SVG 图。
- README 和 Release 文档整理为更适合首次了解项目的结构。

## Notes

- 当前仍以 Tree Layout 为主。
- Timeline、Metro、CLI、Local Git Source 将在后续版本继续推进。

# Treebranch Mark v0.1.0

`v0.1.0` moves Treebranch Mark from the alpha milestone to the first usable release. Compared with `v0.1.0-alpha`, this release focuses on usability, stability, and the first real visual output.

## Highlights

- Added Source-level Memory Snapshot Cache to reduce repeated GitHub API requests.
- Added GitHub contributor avatars for SVG commit nodes.
- Added circle fallback when an avatar is unavailable.
- Improved avatar URL handling to avoid empty avatar URLs and invalid SVG output.
- Browser Viewer can now display SVG generated by the full rendering pipeline.
- Refined README and release documentation for faster project understanding.

## Notes

- Tree Layout remains the only supported layout in this release.
- Timeline, Metro, CLI, and Local Git Source are planned for future versions.
