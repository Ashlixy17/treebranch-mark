# v0.1.0-alpha Release Checklist

This checklist is the release gate for `v0.1.0-alpha`.

Do not create the Git tag until every required item is checked.

## 1. Code Scope

- [x] Source Layer is complete for the MVP.
- [x] Commit Parser is complete for the MVP.
- [x] Branch Graph Builder is complete for the MVP.
- [x] Tree Layout is complete for the MVP.
- [x] RenderModel is complete for the MVP.
- [x] SVG Renderer is complete for the MVP.
- [x] Browser Viewer calls `RenderPipeline`.
- [ ] Optional GitHub Token support is implemented.
- [ ] Source-level memory snapshot cache is implemented.
- [ ] GitHub `401` responses map to `bad-credentials`.

## 2. Security

- [ ] Token is configured only through `GitHubRestClientOptions`.
- [ ] Token does not enter `GitSourceInput`.
- [ ] Token does not enter `GitSourceSnapshot`.
- [ ] Token does not enter `RenderPipeline.render(input)`.
- [ ] Token does not enter `RenderPipelineResult`.
- [ ] Token does not enter Parser, Graph Builder, Layout, RenderModel, or Renderer.
- [ ] Token is not written to `localStorage`.
- [ ] Token is not written to `sessionStorage`.
- [ ] Token is not added to the URL.
- [ ] Token is not shown in Snapshot JSON debug output.
- [ ] Token leakage protection is covered by automated tests.

## 3. Cache

- [ ] Cache belongs to Source.
- [ ] Pipeline does not know cache exists.
- [ ] Parser, Graph Builder, Layout, RenderModel, and Renderer do not know cache exists.
- [ ] `MemorySourceCache` or equivalent in-memory snapshot cache exists.
- [ ] Cache key includes owner, repo, branch, max commits, and include flags.
- [ ] Cache key does not include GitHub Token.
- [ ] `cache: 'reload'` bypasses existing cached data.
- [ ] `cache: 'no-store'` bypasses cache reads and writes.
- [ ] Cache behavior is covered by automated tests.

## 4. Browser Demo

- [ ] `npm run dev` starts successfully.
- [ ] Browser opens the local app.
- [ ] `Ashlixy17/treebranch-mark` can generate an SVG.
- [ ] Optional: `octocat/Hello-World` can generate an SVG.
- [ ] Optional: `vuejs/core` can generate an SVG.
- [ ] SVG preview contains commit nodes.
- [ ] SVG preview contains commit edges when the history has multiple commits.
- [ ] Loading state works.
- [ ] Error state works.
- [ ] Invalid token shows a focused authentication message.
- [ ] Authentication error message is available in English, Simplified Chinese, and Japanese.
- [ ] Demo screenshot is saved to `docs/assets/v0.1.0-alpha-demo.png`.
